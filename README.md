# 🎓 CGPA Calculator

A responsive **CGPA Calculator** designed in **Figma** and developed using **HTML, CSS, and JavaScript**.

## ✨ Features

- 📚 Semester-wise SGPA Calculation
- 🎯 CGPA Calculation
- 📊 Percentage Calculation
- 📌 Total Credits Calculation
- ❌ Backlog Tracking
- 💾 Save Semester Data using Local Storage
- 📱 Responsive User Interface

## 🛠️ Tech Stack

- Figma
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- GitHub Pages
